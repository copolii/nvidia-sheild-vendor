KNOWN ISSUES

-       MBR support is removed from the boot loader release. Hence this bootloader works only with K-35 Google releases.

-   User Partition size is increased to ~30GB and it works with kernel that supports the larger userdata ( HRG34 and above )

-  Supports all portable and mortable with 1G

IMPORTANT NOTES

Upgrading from 1.00.15 release  to this boot loader via  Fast Boot Flashing: 

1. fastboot flash motoboot moto_boot.bin         

2. Flash the other images (system, boot ,etc )

Upgrading from  1.00.10 and prior release  to this boot loader via  Fast Boot Flashing: 

1. fastboot flash motoboot  moto_boot.bin (1.00.15)      : Upgrade to 1.00.15 bootloader before upgrading to 1.00.16

2. fastboot reboot-bootloader                             : The device will reboot in to fastboot mode

3. fastboot flash motoboot moto_boot.bin            : Flashing this moto_boot image will update the new partition table

4. Flash the other images (system, boot ,etc )

 

Note: Once the boot loader is upgraded to this version don’t use the fast boot down grade the boot loader to 08, NVflash needs

To be used downgrade the boot loader alone with appropriate .cfg .bct and .bat file

Note: The step1 needs to be executed prior to flashing the other images , if it is not followed the device could be bricked.

NVFlash mode for portable :

 

WARNING: NVflash will erase the PDS , Please follow the steps below to back up and restore the PDS .

- Install nvFlash tool located at (http://compass.mot.com/go/347648079)

- Down load the boot loader images ( bootloader.bin,everest_flash_v3.bct, everest_flash_v2.cfg)

- Down load the OS images (system.img and boot.img )

- Down load the everest_nv_flash.bat**

- Connect FACTORY cable to phone and PC.

- Power up the phone and select the NVFlash mode on the menu. 

- Run the everest_nv_flash.bat file.

- Backup the PDS Partition by choosing optinon "b".

- After fully flashing the device by choosing optinon "f"

- restore the PDS Partition by selecting option "p".

 

NVFlash mode for mortable :

 

- Install nvFlash tool located at (http://compass.mot.com/go/347648079)

- Down load the boot loader images ( bootloader.bin,everest_flash_v3.bct, everest_flash_v2.cfg)

- Down load the OS images (system.img and boot.img )

- Down load the everest_nv_flash.bat**

- Put the phone in RCM mode ( Press "AP FF" button and press "AP RSOUT" button, then release "AP RSOUT" , then release "AP FF" button ).

- Connect FACTORY cable to phone and PC.

- Run the everest_nv_flash.bat file

- Backup the PDS Partition by choosing optinon "b".

- After fully flashing the device by choosing optinon "f"

- restore the PDS Partition by selecting option "p".

 

**Note: that each invocation of everest_nv_flash.bat script requires target to be reset manually into NVFLASH mode first.
